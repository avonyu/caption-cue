# CaptionCue - 幕听 HTML 原型

这是基于 `prototype/APP_FLOW_PROPOSAL.md` 转成的静态 HTML 设计稿。

## 打开方式

1. 直接用浏览器打开 `prototype/index.html`。
2. 左侧导航可切换抖音学习模式、运行流程、抖音适配、AI 模型和 MVP 范围。
3. 抖音学习模式页可点击底部工具栏中的“幕听”图标打开菜单，再点击“学习模式”切换到沉浸式学习布局。
4. 学习模式中的右侧字幕行可点击，模拟字幕跳转和当前句更新。

## 当前原型覆盖

1. CaptionCue - 幕听的产品结构：Plasmo + TypeScript + React + Zustand + Tailwind CSS。
2. 抖音 `https://www.douyin.com/*` 页面适配逻辑。
3. 无字幕时音频提取、AI 转写、WebVTT 应用流程。
4. DeepSeek 等指定大模型配置和任务路由。
5. MVP 第一阶段交付范围。
6. 单词级学习交互：悬浮单词暂停视频，点击单词显示音标、读音、释义和例句。
7. 字幕显示模式：纯英文、纯中文、中英双语三种模式。
8. 抖音视频页底部工具栏入口、深色菜单弹窗和学习模式布局。
